When you have gotten a character card, the actions of the skills(except KKGs) which are on your card can be found here. Skills on this page can also be bought using XPs. The XPs of skills are stated at the end of the move.

TYPES OF SKILLS

(nom): Normal Skill

(atk): Attacking Skill

(time): Time-Based Skill

(def): Defensive Skill

(neg): Neglecting Skill

(gain): Gain Skill

Note: A card slot can only hold one skill, but when the slot holds a mode, it can contain 3 modes.

You can only use a particular skill once, unless it is stated in the description of the skill that it can be used more than once.

A skill can be wasted, which means, if the move is used in a wrong situation, the move is counted but will have no effect in the battle.

To use a skill against a player, just type the skill name.

A move can be sold for half the XPs it is worth.
