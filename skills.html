Skills & Weapons


All Saiyan, Ninja, Wizard & Quirk moves, as well as weapons


When you have gotten a character card, the actions of the skills(except KKGs) which are on your card can be found here. Skills on this page can also be bought using XPs. The XPs of skills are stated at the end of the move.


Types of Skills


There are 6 different type of skills, the skill type is stated in a bracket after the name of the skill. The 6 types are as follows;


(nom): Normal Skill: These skills only take effect when a certain condition is met.


(atk): Attacking Skill: These skills give damage to players.


(time): Time-Based Skill: These are skills that have effects which last for more than one turn.


(def): Defensive Skill: These are skills which prevent damage from hitting you.


(neg): Neglecting Skill: These are skills which end the effect of other skills.


(gain): Gain Skill: These skills are used to help a player gain specific things, such as skills, health, charge points, etc.


Note: A card slot can only hold one skill, but when the slot holds a mode, it can contain 3 modes.


If a turn is skipped you lose 2hp but gain 2cp.


You can only use a particular skill once, unless it is stated in the description of the skill that it can be used more than once.


A skill can be wasted, which means, if the move is used in a wrong situation, the move is counted but will have no effect in the battle.


To use a skill against a player, just type the skill name.


A move can be sold for half the XPs it is worth.

Weapons


Weapons are kept in your skill card, a slot can hold up to 3 weapon skills. Wepons are mainly used to conserve cp while still maintaining an attacking mindset.

To use a weapon, type the weapon name followed by ''equip'' and then the weapon skill.

If you then want to use the same weapon, if it is still equipped, there is no need to type eqiup again.

If a weapon is unequipped, you have to use a turn to equip it, only 1 weapon can be equipped at a time.

Weapons can be bought by any race.
